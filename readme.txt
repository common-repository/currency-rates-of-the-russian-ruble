Плагин с курсами валют ЦБ РФ. Обновление в реальном времени.

== Description ==

Плагин с курсами валют ЦБ РФ относительно российского рубля.
Данные обновляются в момент запроса и кэшируются.
Чтобы вывести курсы валют программировать не нужно, 
достаточно воспользоваться шорткодами.

Источник данных [https://www.cbr-xml-daily.ru ](https://www.cbr-xml-daily.ru "").

== Installation ==

== Installing manually: =

1. Разархивируйте все файлы в папку `/wp-content/plugins/cur-rates-cb`
2. Войдите в админ панель WordPress и активируйте плагин '	
Курсы валют российского рубля' через меню 'Плагины'
3. Список поддерживаемых валют можно посмотреть в настройках: меню 'Настройки'-> 'Курсы валют рубля'
